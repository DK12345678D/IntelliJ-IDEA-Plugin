This IntelliJ IDEA plugin highlights TODO comments in Kotlin files and displays them in a side panel.

🎯 Features:

Inline highlight of TODO comments in Kotlin

Tool Window listing all TODOs with line info

Click to navigate to source

Manual parsing (no external libraries)

🧰 Built With:

Java 17

IntelliJ Platform SDK 2024.1

Gradle Plugin Framework

🚀 How to Run:

Clone the repo

Open in IntelliJ IDEA

Run the Gradle task: runIde
